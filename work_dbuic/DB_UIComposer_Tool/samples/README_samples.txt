DB_UIComposer samples / tutorial

[v0.4.65+sample-manager]
サンプル管理を「基礎編 / 応用編」に分割しています。
基礎編03と応用編は、データベース代入を中心に再構成しています。

■ 基礎編
- 基礎編 01：シーン/グループ/ウィンドウの概念
- 基礎編 02：主要パーツの用途
- 基礎編 03：データベース代入
  （アクター / アイテム / 用語 / 所持金 / ゲージの現在値・最大値参照）

■ 応用編
- 応用編 01：オリジナルステータス画面（アクター・用語・所持金の代入）
- 応用編 02：オリジナル選択肢UI（会話＋選択肢＋アイテム詳細の代入）
- 応用編 03：アクションゲーム風UI（画面端HUD＋スキル/敵/状態の代入）

■ サンプル管理（ツール上部ボタン）
- 読込: 基礎編/応用編サンプルを現在のレイアウトへ追加
- 現在シーンを新規サンプル保存: 編集後のサンプルを新規登録
- 現在シーンを紐付サンプルへ上書き: 読み込んだサンプルを編集結果で更新
- 初期化/削除: サンプルを初期状態へ戻す（またはカスタム削除）
- サンプル編集を全初期化: すべてのカスタム保存を削除して初期状態へ戻す

※ 初期サンプル本体（バックアップ）は保持されます。壊しても初期化できます。
※ 旧「保存/読込と再利用」「戦闘HUD」などを上書き保存していた場合は、
  『初期化』または『サンプル編集を全初期化』で新しい初期サンプルへ戻せます。

■ 画像差し替えについて
応用編サンプルには、画像未準備でも進められるように
「画像差し替えスロット（imageパーツ）」を配置しています。
fileName が空のスロットへ、後から画像を割り当ててください。
